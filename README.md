# Smart Employee Onboarding & Identity Service

## Project Overview

A serverless Employee Onboarding application built using AWS services.

## AWS Services Used

- AWS Lambda
- Amazon DynamoDB
- Amazon API Gateway
- Amazon Cognito
- Amazon S3
- AWS Step Functions
- Amazon SES
- Amazon CloudWatch
- IAM
