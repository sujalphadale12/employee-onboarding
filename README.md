[200~# 🚀 Smart Employee Onboarding & Identity Service

A cloud-native serverless Employee Onboarding application built on **AWS** that automates employee registration, document management, approval workflows, and secure document access using Amazon S3 pre-signed URLs.

---

## 📌 Project Overview

This application enables organizations to onboard employees securely through a web interface.

Employees can be registered, reviewed, approved or rejected, upload onboarding documents, and securely download them using temporary pre-signed URLs generated by AWS Lambda.

---

## ✨ Features

- Employee Registration
- Employee Management Dashboard
- Employee Profile View
- Employee Approval & Rejection
- Secure Document Upload
- Secure Document Download
- Amazon S3 Pre-signed URLs
- DynamoDB Metadata Storage
- Serverless Architecture
- REST API Integration
